12 · React Component Inventory
Layout & Navigation
AppShell
The primary layout wrapper containing the Sidebar and Topbar. Handles page titles and action buttons.
Sidebar
The navigation panel providing links to Dashboard, Projects, Settings, and Design System.
Breadcrumbs
Utility for path-based navigation (e.g., My Projects / Workspace / Decision).
PresentationMode
Fullscreen chronological walkthrough for presenting decisions to clients.
Information Display
Metric
Stat block used for displaying numeric indicators (e.g., Total Decisions, This month).
FlowStep
Numbered step cards used to explain the application process on the Overview page.
EmptyState
Standardized placeholder for lists or boards containing no items, prompting user action.
SectionBlock / ListBlock
Content grouping blocks for cleanly rendering description paragraphs or bulleted lists (Advantages).
Interactive Form Elements
Field / SelectField
Generic input controls with labels and helper text handling text boundaries.
ListEditor
Dynamic input array builder (used for adding/removing items in form lists).
ImageField
Flexible media ingestion supporting direct URL, local file uploads, and clipboard pasting.
StatusRadioGroup
Selection pills for picking decision statuses with color coordination logic built-in.
Modals & Overlays
ModalFrame
Base container providing backdrop, title bar, close action, and inner padding constraints for all dialogs.
ProjectModal & DecisionModal
Task-specific dialog components managing the state for initializing or updating records.
